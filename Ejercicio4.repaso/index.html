Crea un código javascript que haga lo siguiente: Dirá al usuario que introduzca un número positivo (indicando que con 0 se termina la ejecución). Si el número introducido es negativo se ignorará. Se irá preguntando al usuario que introduzca números hasta que se introduzca un 0. Si el número es 0 se terminará la ejecución y se mostrará cuántos números se han introducido, cuál es el mayor número introducido, cuál es el menor, cuál es la suma y cuál es la media de todos los números introducidos.
